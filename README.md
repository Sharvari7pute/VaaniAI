
 🎓 VaaniAI

VaaniAI is an AI-powered study companion designed to help students learn smarter, not harder.

It combines artificial intelligence with modern web technologies to provide an interactive learning experience through PDF analysis, AI tutoring, quizzes, flashcards, personalized study plans, and more.

---

 ✨ Features

- 🤖 AI Study Assistant
- 📄 Chat with PDFs
- 📝 AI Notes Generator
- 🎯 Quiz Generator
- 🧠 Flashcards
- 📅 AI Study Planner
- 📊 Learning Dashboard
- 🔐 Secure Authentication
- 🌙 Dark Mode
